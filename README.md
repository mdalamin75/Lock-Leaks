# Leaked Content Removal Landing Page

A pixel-perfect Next.js landing page built with Tailwind CSS, featuring a modern header and hero section.

## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `app/` - Next.js app directory with pages and layouts
- `components/` - React components (Header, Hero)
- `app/globals.css` - Global styles with Tailwind CSS
- `tailwind.config.js` - Tailwind CSS configuration

## Features

- ✅ Responsive design
- ✅ Pixel-perfect header with navigation
- ✅ Hero section with radar-style graphic
- ✅ Platform icons (Google, Reddit, Instagram, Twitter/X, etc.)
- ✅ Modern UI with Tailwind CSS

## Build for Production

```bash
npm run build
npm start
```
